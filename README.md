# Dynamic Pathfinding Agent (NUCES-CFD)

## Description
This project implements A* and GBFS pathfinding algorithms using Pygame.
It includes dynamic obstacle generation and heuristic switching.

## Features
- A* Search
- Greedy Best First Search
- Manhattan & Euclidean heuristics
- Dynamic obstacle mode
- Performance metrics dashboard

## Installation

1. Clone the repository:

git clone https://github.com/MuhammadAbdullah31/dynamic-pathfinding-agent.git

2. Install dependencies:

pip install pygame

3. Run the project:

python main.py

## Controls

Left Click → Set Start, Goal, Obstacles  
Right Click → Remove node  
SPACE → Run algorithm  
A → Switch Algorithm  
H → Switch Heuristic  
D → Toggle Dynamic Mode  
R → Random obstacles  
C → Clear Grid
